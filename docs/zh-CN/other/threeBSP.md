# ThreeBSP

使用 ThreeBSP 库里可以进行现有模型的组合，分别是:差集(相减)、并集(组合、相加)、交集(两几何体重合的部分)

| 名称    | 描述                |
| ------- | --------------------------- |
| intersect（交集） | 使用该函数可以基于两个现有几何体的重合的部分定义此几何体的形状。 |
| union（并集） | 使用该函数可以将两个几何体联合起来创建出一个新的几何体。 |
| subtract（差集） | 使用该函数可以在第一个几何体中移除两个几何体重叠的部分来创建新的几何体。 |
