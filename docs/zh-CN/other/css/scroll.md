# scroll-behavior
> 当用户手动导航或者 CSSOM scrolling API 触发滚动操作时，为一个滚动框指定滚动行为。

在滚动容器上设置了如下的 CSS，实现平滑滚动。
```css
scroll-behavior: smooth;
```
**引用** https://developer.mozilla.org/en-US/docs/Web/CSS/scroll-behavior
